# Ledgerly

> A modern, mobile-first Personal Lending & Relationship Ledger built with Laravel 13, React, and TypeScript.

## Overview

Ledgerly helps users manage financial relationships rather than simply recording debts.

Instead of treating loans as isolated transactions, Ledgerly organizes every interaction around a **Contact**, creating a complete financial timeline that includes:

* Loans
* Repayments
* Financial Activities
* Reminders
* Notes

The goal is to provide a simple, trustworthy, and fast experience for tracking personal lending.
